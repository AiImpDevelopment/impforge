## What is ImpForge?

ImpForge replaces your entire software stack with **one native desktop app**. Chat with AI, write documents, manage spreadsheets, create presentations, automate workflows, manage social media, collaborate with your team — all **offline-first** and **100% private**.

No cloud required. No subscriptions for core features. No data leaves your machine.

> **"Like having Microsoft 365 + Slack + Adobe + Zapier + Cursor in one app — but it runs on your laptop without internet."**

---

## Features

### AI Chat with Smart Routing
Chat with 100+ AI models. ImpForge automatically selects the smallest model that can handle your question — saving costs and keeping things fast.

- **Ollama** — run models locally, 100% private
- **OpenRouter** — 28+ free cloud models as fallback
- **Cascade Inference** — local first, cloud only if needed
- **Inner Thoughts** — AI thinks proactively and suggests helpful actions

### Sovereign Office Suite
A complete Microsoft Office + Adobe replacement, built from scratch:

| Module | Replaces | Key Feature |
|--------|----------|-------------|
| **ForgeWriter** | Word / Google Docs | AI writing assistant with 7 actions |
| **ForgeSheets** | Excel / Google Sheets | Natural language formulas + Agentic Cells |
| **ForgeSlides** | PowerPoint | AI presentation generator |
| **ForgePDF** | Adobe Acrobat | AI summarize, Q&A, convert |
| **ForgeCanvas** | Notion / Figma | 3-panel AI document workspace |
| **ForgeNotes** | Notion / Obsidian | Wiki-links + Knowledge Graph |
| **ForgeCalendar** | Google Calendar | ICS import + AI daily briefing |
| **ForgeMail** | Outlook / Gmail | AI compose with 4 tones |

### Business Automation
- **ForgeFlow** — visual workflow builder (replaces n8n, Zapier, Make)
- **Social Media Hub** — post to 6 platforms with AI content generation
- **Freelancer Hub** — clients, invoices, proposals, time tracking
- **Auto-Publisher** — browser automation for any platform

### Team Collaboration
- **ForgeTeam** — P2P team management with invite codes
- **ImpBook** — shared AI knowledge workspace
- **Team Chat** — real-time messaging
- **Team Goals** — milestones with progress tracking

### Developer Tools
- **CodeForge IDE** — 15-panel IDE with LSP, Git, Debug, Terminal
- **ForgeAI** — AI code completion, edit, explain, fix, test
- **Docker Dashboard** — container management
- **GitHub Integration** — repos, issues, PRs

### SwarmForge — Productivity RPG
Your work powers a medieval fantasy idle RPG:
- **20 unit types** with visual evolution
- **18 bio-ships** for space colonization
- **22 buildings** with OGame-style upgrades
- **240+ mutations** — every 5 levels, your units visually change
- **Achievement system** with 40 achievements, levels, and streaks
